## 📖 About The Project

**SunCart** is a modern e-commerce web application focused on summer essentials. From premium sunglasses to beach accessories, skincare products to trendy summer outfits – SunCart brings the sunshine to your doorstep. Built with cutting-edge technologies, this project demonstrates a complete e-commerce solution with seamless user experience, beautiful animations, and robust authentication.

### 🎯 Key Features

| Feature | Description |
|---------|-------------|
| 🛍️ **Product Catalog** | Browse through 12+ summer products with detailed information |
| 🔐 **Authentication** | Secure login/signup with BetterAuth + MongoDB |
| 🎨 **Modern UI** | Beautiful gradient designs with smooth animations |
| 📱 **Responsive** | Mobile-first design that works on all devices |
| 🛒 **Product Details** | Dynamic product pages with image gallery |
| 💳 **Cart System** | Add to cart, quantity updates, and checkout flow |
| ⭐ **Rating System** | Visual star ratings for products |
| 🏷️ **Stock Management** | Real-time stock status and availability |
| 🔍 **Quick View** | Hover-based product preview |

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or later)
- [npm](https://npmjs.com/) (v9 or later)
- [MongoDB](https://mongodb.com/) (local or cloud instance)

### Installation

Follow these steps to get your development environment set up:

```bash
# 1. Clone the repository
git clone git@github.com:najimhaq/suncart-summer-essentials-store.git
cd suncart

# 2. Install dependencies
npm install

# 3. Set up environment variables
# Create a .env.local file with your MongoDB URI
echo "MONGODB_URI=your_mongodb_connection_string" > .env.local

# 4. Run the development server
npm run dev

🚀 Deployment
Vercel (Recommended)
npm install -g vercel
vercel
